# План разработки (итерации)

## Отчет о прогрессе

| Итерация | Статус | Иконка | Итог/заметка |
|---|---|---|---|
| 1 | Готово | ✅ | Каркас + главная страница |
| 2 | Готово | ✅ | Создание и список заказов |
| 3 | Готово | ✅ | Исполнитель: взять и завершить |
| 4 | Готово | ✅ | Админ и оплата (заглушка) |

## Итерации (KISS)

### 1) Базовый скелет и главная страница
- [x] Создать каркас приложения, страницу приветствия, базовый запуск.
- [x] Проверка: приложение открывается в браузере.

### 2) Заказы: создание и список
- [x] Формы и страницы: создать заказ, список заказов.
- [x] Проверка: заказ создается и отображается в списке.

### 3) Исполнители: выбор и выполнение
- [x] Возможность исполнителю взять заказ и отметить выполнение.
- [x] Проверка: статус заказа меняется по действиям исполнителя.

### 4) Админ и оплата (заглушка)
- [x] Админ меняет статусы и отмечает оплату вручную.
- [x] Проверка: админ видит все заказы и может фиксировать оплату.
